Calendar

This is a collection of upcoming events with potential application to cloud physics, including workshops and trainings that aren't cloud-focused. If you have a suggested addition, send it to uk-early-career-cloud-physics@gaggle.email – anyone can post to this list, even if you're not subscribed.

If you have missed a registration deadline, it is always worth emailing the organisers and asking politely whether late registration is possible, especially for events like hackathons where the organisers want active participants.
